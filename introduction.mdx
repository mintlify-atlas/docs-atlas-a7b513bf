## What is Generative AI for Krita?

Generative AI for Krita is a plugin that brings powerful AI image generation directly into [Krita](https://krita.org), the professional digital painting application. Instead of switching between separate tools, you can generate, inpaint, and refine images without leaving your painting workflow.

The plugin connects to a [ComfyUI](https://github.com/comfyanonymous/ComfyUI) backend — either managed locally, a custom installation, or a cloud service — and gives you precise control over AI generation through Krita's native layer and selection tools.

## Supported hardware

<Note>
  For local generation, a GPU with at least **6 GB VRAM** is recommended. Generation on CPU is possible but very slow.
</Note>

| Hardware | Support |
|---|---|
| NVIDIA GPU | Fully supported via CUDA (Windows/Linux) |
| AMD GPU | Supported, may require custom ComfyUI setup |
| Apple Silicon | Supported via MPS on macOS 14+ |
| Intel XPU | Supported (Windows/Linux), may see performance issues |
| CPU | Supported but very slow |

## Getting started

<Steps>
  <Step title="Install Krita">
    Download and install [Krita 5.2.0 or newer](https://krita.org/en/download/) for your platform.
  </Step>
  <Step title="Install the plugin">
    Follow the [Plugin Installation Guide](/installation) to download and install the plugin in Krita.
  </Step>
  <Step title="Set up a server">
    Choose your generation backend in the [Server Setup Guide](/server/overview): cloud service, local managed server, or custom ComfyUI.
  </Step>
  <Step title="Start generating">
    Read the [Basics Guide](/guide/basics) to create your first AI-generated image.
  </Step>
</Steps>
